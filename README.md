# Event List

## Installation

prerequisite:

NodeJS:

```
https://nodejs.org/en/download/
```

---

### Front-end

```
cd front
npm install
npm run build
npm run serve
```

---

### MongoDB

#### Using docker

```
cd db
docker-compose up
```

#### Using local mongoDB server

```
https://docs.mongodb.com/manual/installation/
```

---

### Back-End

```
cd back
npm install
npm run build
npm run start:prod
```

## Running tests

### Back-end

```
cd back
npm run test
```

### Front-end

```
cd front
npm run test
```
